WORDPRESS

Wordpress

- это современная площадка для создания сайтов
в интернете, такие площадки называются CMS. У неё
большой набор функций, таких как:
-Возможность сделать сайт, выбрав
любую из понравившихся тем в их каталоге.
-Возможность создания своих тем.
-Возможность добавить свой плагин
(для профессионалов)
или установить один из имеющихся
-Также можно воспользоваться
удобным конструктором

Поддержка проектов

Поддержка проектов.

После создания сайта, логотипа, дизайна
и ботов у нас есть поддержка которая
длится 2 месяца. После чего
вы можете продлить его на любой срок из:
-2 месяца
-4 месяца
-6 месяцев
Под поддержкой подразумевается то, что
мы вам будем доделывать
ваш проект в течении срока
действия поддержки
